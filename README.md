This is a Java Springboot application.

In order to clone and run this application locally, you need to have the following installed on your machine
- Java21
- Any IDE (IntelliJ recommended)
- PostgreSQL DB and TimeScale DB (I had installed on my local using a docker image). I had created and tested data in the PostgreSQL database locally.
- I had used OpenAI chat model and saved the Api key on my machine's environment variable. Rhis repo will not have it as a sensitive information. You could do the same (Generate and add Api key from OpenAI) to run it locally.
